# Backnd CLI 한국어 매핑

사용자가 한국어로 요청할 때 CLI 개념을 추론하기 위한 힌트 테이블.

**사용법**:
1. 요청에서 도메인 키워드와 액션 키워드를 식별한다
2. 각각의 힌트 단어를 얻는다
3. `backnd --help`로 현재 CLI 구조를 확인하고, 힌트와 가장 가까운 명령어를 찾는다
4. 실제 경로는 항상 `--help`로 발견한다 — 이 매핑의 값을 그대로 CLI에 넘기지 않는다

## 주의사항

- 힌트는 `backnd --help` 탐색의 출발점이다. 실제 명령어명과 다를 수 있다.
- CLI 버전이 올라가면서 서브커맨드가 최상위 명령어로 승격되거나, 명령어가 합쳐지거나 이름이 바뀔 수 있다. 힌트로 방향만 잡고, 실제 패스는 항상 `--help`로 확인한다.
- 매핑 파일 기준: 2026-04-16 (backnd v0.1.0)
